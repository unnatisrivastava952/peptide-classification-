User Manual

Welcome to the Peptide Classification Web App! This guide will help you understand how to use the application effectively.

1. Getting Started

To classify a peptide sequence, follow these steps:

  • Enter a valid peptide sequence (e.g., ACDEFGHIKLMNPQRSTVWY).
  • Click the Predict button.
  • View the classification result, confidence score, and other details.

2. Input Guidelines

Make sure your peptide sequence contains only valid amino acids (A, C, D, E, F, G, H, I, K, L, M, N, P, Q, R, S, T, V, W, Y).

3. Understanding Results

After submitting the sequence, the model will predict its category:

  • Antibacterial - Peptide with antibacterial properties.
  • Antiviral - Peptide effective against viruses.
  • Antimicrobial - Broad-spectrum antimicrobial peptide.
  • Antifungal - Peptide with antifungal properties.

4. Troubleshooting

If you encounter issues, try the following:

  • Ensure the sequence contains valid amino acids.
  • Check if the model and scaler files are correctly loaded.
  • Reload the app and try again.

5. Contact Support

If you need further assistance, reach out to the team at support@example.com.
